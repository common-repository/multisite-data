Enable Multisite Data plugin that serves the purpose of viewing data of multiple websites on a single platform.

== Description ==

This plugin serves the purpose of viewing data of multiple websites on a single platform.

What it includes:

1. Dashboard
2. Viewing Websites
3. Viewing Products on all the websites
4. View the Ordered Product List from all of the websites
5. View list of Customers on all the websites
6. View the Sales Report
7. View Order Status
8. Advance Search Order 



== Installation ==

1.  Upload your plugin folder to the '/wp-content/plugins' directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
